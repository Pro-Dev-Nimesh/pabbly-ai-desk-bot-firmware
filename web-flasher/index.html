Flash your device from the browser.

Plug your XIAO ESP32-S3 into this computer with a USB-C cable, then click the button. No Arduino IDE needed — this installs the firmware directly over USB.

Your browser can't flash devices. Use Chrome or Edge on a desktop. You need to visit this page over https.
Chrome / Edge · desktop only · uses WebSerial

After flashing

  1. The device shows "Join 'VoiceBuddy-Setup'" — connect to that Wi-Fi on your phone.
  2. Pick your home Wi-Fi and confirm the Server URL + Device Token.
  3. It shows a 6-digit code — enter it in the console to claim the device.
  4. Touch the pad and talk. That's it.
If the button says "no binaries yet": the firmware is built automatically by GitHub Actions. Push the repo and let the Deploy web flasher action finish once — it compiles the firmware and places the .bin files next to this page.
